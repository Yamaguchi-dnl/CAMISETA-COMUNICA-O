# **App Name**: IAP Camisetas

## Core Features:

- Product Catalog Display: Showcase available t-shirts on the homepage in a responsive grid, with images, names, brief descriptions, and prices for quick browsing.
- Individual Product Pages: Each t-shirt will have a dedicated sales page, presenting full details, multiple views, price, and options for size and quantity selection.
- Customer Order Form: An intuitive form on each product page allows customers to input their contact information, desired payment method (Pix/Parcelado), and any special observations.
- Firestore Order Persistence: Upon form submission, securely save all entered order and customer details into a designated collection within Firebase Firestore.
- WhatsApp Direct Integration: Automatically generate a pre-filled WhatsApp message using customer order details and specific payment instructions, then redirect the user directly to WhatsApp to finalize the purchase.
- Responsive and Mobile-First Design: Ensure the entire application layout and components are fully responsive and optimized for a seamless user experience across all devices, prioritizing mobile viewing.

## Style Guidelines:

- Primary color: A sophisticated charcoal black (#111111) defining key structural elements and typography, creating a strong, modern visual anchor.
- Background color: A clean, ethereal light gray (#f5f5f5) acting as a minimalist canvas, enhancing content visibility and contributing to an airy, organized feel.
- Accent color: A vibrant and unmistakable WhatsApp green (#25D366), strategically used to highlight crucial calls to action and emphasize the direct communication integration.
- All text uses 'Inter', a versatile sans-serif font chosen for its modern, clean, and objective aesthetic, ensuring optimal readability across all sections and devices.
- Emphasize clean, line-based icons that convey information clearly and concisely, aligning with the application's modern and minimalist visual language, especially for form fields and navigation.
- A mobile-first, grid-centric layout on the homepage offers intuitive product discovery, while individual product pages feature prominent imagery and a vertically optimized, step-by-step order form for clarity.
- Subtle and sophisticated animations, such as gentle fades for content transitions and delicate hover effects on interactive elements, provide a polished and responsive user experience without being distracting.
